## Claude Code 투입 팁

1. 세션마다 `CLAUDE.md` + 해당 태스크의 문서 섹션(예: "TAD §6과 T1.3을 구현해줘")을 지정
2. 태스크 완료 기준은 AC 그대로 요구: "AC를 만족하는 테스트를 먼저 작성하고 구현해"
3. M1이 전체 리스크의 80% — M1.4까지는 UI 없이 최소 테스트 페이지로 검증하고 넘어갈 것
4. 생성 품질 판단(에센스/LoRA 톤)은 사람이 직접 평가하고, 결과를 프리셋 성공 기준 메모에 기록해 자산화
