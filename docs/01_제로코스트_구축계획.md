## 1. 원칙

1. **금전 비용 0원**: 유료 API, 유료 라이선스, 클라우드 GPU 결제, 외주 없음. 필요한 것은 본인 맥(Apple Silicon)과 시간뿐
2. **전부 오픈소스/무료 배포물**: 모델 가중치는 Hugging Face에서 무료 다운로드, 도구는 MIT/Apache/GPL 계열
3. **상용 이용 가능 라이선스만**: 나중에 유료화하더라도 문제없도록 처음부터 라이선스 확인

## 3. 아키텍처 (무료 스택 기준)

```
┌─────────────────────────────────────────────┐
│ Tauri 앱 (프론트: React/Svelte — 무료)         │
│  생성 UI · 갤러리 · 프리셋/에센스 편집기 · 설정   │
├─────────────────────────────────────────────┤
│ Rust/JS Core: 작업 큐 · 프리셋 조립 ·          │
│  한→영 변환 · SQLite 히스토리                  │
├────────────────────┬────────────────────────┤
│ 추론: ComfyUI 헤드리스│ 학습: kohya sd-scripts   │
│ (localhost API)     │ 캡셔닝: WD14/Florence-2  │
│ SDXL+LoRA+IP-Adapter│ (Python, 서브프로세스)    │
├────────────────────┴────────────────────────┤
│ ~/Library/Application Support/LocalBrush/     │
│  models/ · loras/ · outputs/ · presets.json  │
└─────────────────────────────────────────────┘
```

ComfyUI를 엔진으로 쓰는 이유: 생성·IP-Adapter·업스케일 파이프라인이 이미 구현돼 있어 **개발 공수(=사실상 유일한 비용)를 최소화**. 앱은 ComfyUI API에 워크플로 JSON을 보내는 프론트 역할만 하면 됨.

## 4. 단계별 구축 가이드 (혼자, 0원)

### Step 0 — 하루 만에 개념 검증 (코드 작성 전)
1. ComfyUI 설치 (`git clone` + `pip install`, Apple Silicon 지원)
2. SDXL base + IP-Adapter 가중치 다운로드 (HF 무료)
3. 브랜드 참조 이미지 몇 장으로 IP-Adapter 워크플로 실행 → **에센스 방식으로 톤이 나오는지 눈으로 확인**
4. 대안 검증: Draw Things(무료 맥 앱)로도 동일 실험 가능
→ 여기서 품질이 안 나오면 앱을 만들 이유가 없으므로 가장 먼저 수행

### Step 1 — LoRA 학습 검증 (1~2주)
1. 브랜드 이미지 20~50장 준비, WD14/Florence-2로 캡션 생성
2. kohya sd-scripts로 SDXL LoRA 학습 (M 시리즈에서 수 시간, 밤에 돌리기)
3. Step 0 결과와 비교 → 에센스 vs LoRA 품질 차이 기록 (프리셋 성공 기준의 초안이 됨)
- 학습 속도가 감당 안 되면: SD 1.5 LoRA로 다운시프트(훨씬 빠름). ※ Colab/Kaggle 무료 GPU도 있으나 이미지가 기기를 떠나므로 "완전 로컬" 원칙과 충돌 — 본인 자산이면 선택지로만 인지

### Step 2 — MVP 앱 (4~8주, 사이드 프로젝트 기준)
1. Tauri 프로젝트 생성, 화면 3개: 생성 / 갤러리 / 스타일·프리셋
2. 앱 최초 실행 시: 내장 스크립트가 ComfyUI + 모델을 사용자 기기에 자동 설치(다운로드 진행률 표시)
3. 프리셋 JSON → ComfyUI 워크플로 JSON 변환기 작성 (핵심 개발 작업)
4. 에센스 마법사: Florence-2로 참조 이미지 스타일 서술 생성 → 프리셋 저장
5. SQLite 히스토리, PNG 저장

### Step 3 — 학습 기능 통합 (3~4주)
1. kohya를 서브프로세스로 호출하는 학습 큐 (진행률 파싱, 백그라운드, 완료 알림)
2. 학습된 .safetensors를 스타일 목록에 자동 등록

### Step 4 — 무료 배포
1. `tauri build` → .dmg 생성
2. GitHub Releases에 업로드 (무료), README에 설치 안내
3. Homebrew cask 등록 (무료) — `brew install --cask localbrush`

## 6. 공증(Notarization) 없이 배포하는 법 — $99 아끼기

- 서명 없는 앱은 macOS Gatekeeper가 차단하지만, 사용자가 **우클릭 → 열기** 또는 `xattr -cr /Applications/LocalBrush.app`으로 실행 가능
- README와 배포 페이지에 이 안내를 명확히 게시 (스크린샷 포함)
- 트레이드오프: 설치 마찰 증가, 일반 사용자 배포에는 불리 → **개인/팀 내부용은 무서명으로 충분**, 대중 배포 단계에서만 $99 지출 검토
- Homebrew cask 배포 시 `--no-quarantine` 옵션 안내로 마찰 완화 가능
