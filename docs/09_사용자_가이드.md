# LocalBrush 사용 가이드

이 문서는 개발자가 아니라 **실제로 앱을 쓰는 분**을 위한 안내서입니다. 화면에서 무엇을 누르면 무슨 일이 일어나는지 순서대로 설명합니다.

> 요구 사항: Apple Silicon(M1 이상) Mac. 인터넷 연결은 처음 설치할 때와 새 스타일 도구를 처음 쓸 때만 필요하고, 평소 이미지 생성은 인터넷 없이도 됩니다. 만든 이미지와 입력한 내용은 외부로 전송되지 않고 내 컴퓨터 안에만 저장됩니다.

---

## 1. 설치하고 처음 실행하기

1. [Releases](https://github.com/claire-yunsumin/makesource/releases) 페이지에서 최신 `.dmg` 파일을 내려받습니다. (또는 `brew install --cask claire-yunsumin/localbrush/localbrush`)
2. dmg를 열고 LocalBrush 아이콘을 Applications 폴더로 끌어다 놓습니다.
3. 처음 실행하면 macOS가 "확인되지 않은 개발자" 경고를 띄웁니다. 이 앱은 무료로 배포하기 위해 애플의 유료 공증을 받지 않았을 뿐 안전합니다. Finder에서 앱을 **우클릭(또는 control 키를 누른 채 클릭) → 열기**를 선택하고, 뜨는 창에서 다시 **열기**를 누르면 실행됩니다. (한 번만 하면 됩니다)
4. 처음 실행하면 **설치 화면**이 나옵니다.
   - **표준**(약 10GB, 고품질) 또는 **라이트**(약 4GB, 가벼움) 중 하나를 고릅니다. 내 컴퓨터 메모리를 감지해서 적당한 쪽에 "추천" 표시가 붙어 있으니, 잘 모르겠으면 추천된 쪽을 고르면 됩니다.
   - [설치 시작하기]를 누르면 자동으로 다운로드·설치가 진행됩니다. 창을 닫지 말고 기다려 주세요 (인터넷 속도에 따라 시간이 꽤 걸릴 수 있습니다).
   - 중간에 앱을 껐다가 다시 켜도 이어서 설치되니 걱정하지 않아도 됩니다.
   - 설치 중 문제가 생기면 화면에 원인이 나오고 [로그 열기]로 자세한 기록을 볼 수 있으며, [재시도]로 다시 시도할 수 있습니다.
5. 설치가 끝나면 자동으로 **생성 화면**으로 이동합니다. 이제부터 쓸 준비가 된 것입니다.

앱 왼쪽에는 항상 5개의 메뉴 아이콘이 있습니다: **생성 · 갤러리 · 스타일 · 프리셋 · 설정**. 평소에는 "생성"과 "갤러리"만 오가도 충분합니다.

## 2. 이미지 만들기 (생성 화면)

가장 많이 쓰게 될 화면입니다. 왼쪽 패널에서 조건을 정하고, 오른쪽에 결과가 뜹니다.

1. **스타일** 줄에서 원하는 브랜드 톤을 고릅니다. 아직 만든 스타일이 없으면 "없음"만 있는데, 이 상태로도 바로 생성할 수 있습니다. (스타일 만드는 법은 [4장](#4-내-브랜드-톤-만들기-스타일-화면) 참고)
2. **이미지 타입** 중 하나를 클릭합니다. (예: 배너, 상품 컷 등 — 미리 준비된 프리셋 6종)
3. **키워드** 칸에 만들고 싶은 대상을 한글로 입력합니다. 예: `통나무집`, `빨간 자동차`. 한글은 자동으로 영어로 바뀌어 내부적으로 사용됩니다.
4. 필요하면 **장수**(1~4장)와 **크기**를 고릅니다.
5. **[생성하기]** 버튼을 클릭하거나 키보드 **⌘(Command)+Enter**를 누릅니다.
6. 오른쪽에 진행 상황이 표시되고, 완료되면 결과 이미지가 나타납니다. 생성 중 마음이 바뀌면 **[취소]** 버튼이나 **Esc** 키로 멈출 수 있습니다.

### 결과 이미지로 할 수 있는 것

각 이미지에 마우스를 올리면 하단에 버튼 3개가 나타납니다.

| 버튼 | 하는 일 |
|---|---|
| ♡ / ♥ | 즐겨찾기 등록·해제. 갤러리에서 즐겨찾기만 모아 볼 수 있습니다 |
| ⬇ PNG | 컴퓨터의 다운로드 폴더에 PNG로 저장 |
| 시드 고정 | 같은 시드(난수 값)로 다시 만들어서, 미세하게만 다른 결과를 비교하고 싶을 때 사용 |

### 고급 옵션 (선택)

"고급"을 펼치면:
- **시드**: 비워두면 매번 랜덤, 숫자를 넣으면 그 값으로 항상 같은 결과 재현
- **변환·프롬프트 미리보기**: 입력한 한글이 실제로 어떤 영어 문장으로 바뀌어 쓰이는지 미리 볼 수 있습니다

## 3. 만든 이미지 모아보기 (갤러리 화면)

지금까지 만든 모든 이미지가 최신순으로 쌓입니다.

- 위쪽 검색창에 키워드를 입력하면 필터링됩니다.
- **[♥ 즐겨찾기만]**을 누르면 즐겨찾기한 이미지만 보입니다.
- 이미지를 클릭하면 **상세 창**이 열리고 여기서 할 수 있는 것:
  - 사용했던 설정(프롬프트, 시드 등) 전체 복사
  - 같은 설정으로 다시 생성
  - **PNG / JPG / WebP** 세 가지 형식으로 내보내기 (투명 배경 옵션 포함)
- 이미지를 Finder나 다른 앱(예: 슬랙, 파워포인트)으로 **직접 드래그**해서 바로 붙여넣을 수도 있습니다.

## 4. 내 브랜드 톤 만들기 (스타일 화면)

"스타일"은 생성할 때 브랜드 느낌을 입히는 기능입니다. 두 가지 방법이 있습니다.

### 방법 A — 빠른 스타일 (에센스, 몇 분)

참고 이미지 몇 장(3~10장)만 있으면 됩니다. 학습 과정 없이 바로 스타일이 만들어집니다.

1. 스타일 화면에서 **[+ 새 스타일 (에센스 · 몇 분)]** 클릭
2. 브랜드 느낌을 보여주는 이미지를 3~10장 끌어다 놓기
3. **[스타일 분석하기]** 클릭 — 잠시 기다리면(첫 사용 시 조금 더 걸릴 수 있음) 자동으로 스타일 설명이 만들어집니다
4. 필요하면 설명 문구를 손으로 다듬고, 이름을 지어 저장
5. 이후 생성 화면의 "스타일" 줄에서 이 이름을 골라 바로 사용

### 방법 B — 정밀 학습 (LoRA, 시간이 걸림)

내 브랜드 이미지 20장 이상으로 더 정확한 스타일을 원할 때 씁니다. 대신 수 시간이 걸리므로 컴퓨터를 켜둔 채로 두어야 합니다.

1. 스타일 화면에서 **[+ 정밀 학습 (LoRA · 수 시간)]** 클릭
2. 이미지 20장 이상 끌어다 놓기 → 자동으로 설명(캡션)이 붙습니다
3. 캡션을 확인·수정하고 다음 단계로
4. 이름과 학습 강도(빠름/표준/고품질)를 고르고 학습 시작
5. 진행률·예상 시간·중간 결과 샘플이 표시됩니다. 끝나면 스타일 목록에 자동 등록됩니다

두 방법 모두 만든 스타일은 스타일 화면 카드 목록에서 삭제할 수 있습니다.

## 5. 자주 쓰는 조합 저장하기 (프리셋 화면)

"프리셋"은 이미지 타입별 기본 설정(문구 앞뒤에 붙는 말, 품질 옵션 등)입니다. 보통은 그대로 써도 되지만, 직접 다듬고 싶다면:

- 왼쪽 목록에서 프리셋을 고르고 오른쪽에서 값을 수정한 뒤 **[버전 저장]**
- 예전 버전으로 되돌리고 싶으면 **[이전 버전 보기/복원]**
- 프리셋 두 개를 나란히 비교하고 싶으면 **[A/B 비교]** — 같은 키워드·시드로 동시에 생성해 결과를 좌우로 비교합니다
- **[내보내기/가져오기]**로 프리셋을 파일로 저장하거나 다른 컴퓨터로 옮길 수 있습니다

---

## 6. 설정 화면

| 항목 | 내용 |
|---|---|
| 모델 관리 | 설치된 AI 모델 목록과 각각의 용량 확인 |
| 캐시 | 분석용 임시 모델 캐시를 지워 저장 공간 확보 (필요하면 다시 자동 다운로드됨) |
| 생성 — 안전 네거티브 | 모든 생성에서 항상 피할 내용을 지정 (기본값이 이미 설정되어 있어 보통 건드릴 필요 없음) |
| 언어 | 한국어/영어 전환 (일부 화면은 아직 한국어만 지원) |
| 정보 | 앱 버전과 이 앱이 사용하는 오픈소스 목록 |

## 자주 묻는 질문

**Q. 인터넷 없이도 쓸 수 있나요?**
설치와 새 스타일 도구를 처음 쓸 때(모델 다운로드)만 인터넷이 필요합니다. 그 외 이미지 생성 자체는 오프라인에서도 됩니다.

**Q. 생성형 AI를 쓰는 거면 호출할 때마다 토큰이나 API 사용량이 청구되나요?**
아니요, 청구되지 않습니다. ChatGPT나 여러 AI 이미지 서비스처럼 외부 서버(API)에 요청을 보내고 사용한 만큼 과금되는 방식이 아니라, 처음 설치할 때 AI 모델 파일을 내 컴퓨터에 통째로 내려받아 두고 **내 컴퓨터 안에서 직접 계산해서** 이미지를 만드는 방식입니다. 그래서 생성 버튼을 몇 번을 누르든, 이미지를 몇 장을 만들든 별도로 청구되는 비용이 없습니다. 인터넷 요금 외에는 전기료 정도가 전부라고 보면 됩니다. (다만 그만큼 내 컴퓨터의 메모리·처리 성능을 사용하고, Apple Silicon Mac이 있어야 합니다 — [설치](#1-설치하고-처음-실행하기) 참고)

**Q. 내 이미지가 어디로 전송되나요?**
아니요. 업로드한 이미지, 입력한 키워드, 만들어진 결과 모두 내 컴퓨터 안에만 저장됩니다. 외부로 나가는 통신은 모델을 내려받거나 앱 업데이트를 확인할 때뿐입니다.

**Q. 생성이 너무 느리거나 실패해요.**
메모리가 부족한 경우 앱이 자동으로 크기를 줄이거나 더 가벼운 모델로 다시 시도하고, 그 사실을 화면에 알려줍니다. 그래도 계속 안 되면 설정 화면에서 저장 공간과 모델 설치 상태를 확인해 보세요.

**Q. 만든 이미지를 상업적으로 써도 되나요?**
앱과 대부분의 구성 요소는 상업적 이용이 가능한 라이선스를 쓰지만, 일부 기반 모델(예: Stable Diffusion)은 자체 라이선스 조건이 있습니다. 설정 화면의 "오픈소스 라이선스" 목록에서 각 항목의 라이선스를 확인하세요.
