# 프리셋 프롬프트 연구

버전: v0.1 | 작성일: 2026-07-05 | 상태: 실험 진행 중

기본 프리셋 6종의 프롬프트·파라미터 실험 기록. 코드상 현재값의 원본은 `src-tauri/resources/presets.default.json`이며, **실험으로 값을 바꾸면 JSON의 `version`을 올리고 이 문서에 실험 근거를 남긴다.** (프리셋은 버전 관리되는 팀 자산 — PRD §1.3)

## 실험 방법

- **고정 조건**: 동일 시드 3개(예: 42, 1234, 20260705) × 표준 키워드 3종으로 생성해 비교. 한 실험에서 변수는 하나만 바꾼다 (prefix/suffix/negative/steps/cfg 중 하나)
- **표준 키워드**: `coffee cup`(사물), `rocket launch`(장면), `smiling robot`(캐릭터) — 프리셋 간 비교를 위해 공통 사용
- **판정**: 각 프리셋의 성공 기준(아래 표)을 시드 3개 모두 충족하면 ✅, 일부만이면 ⚠️, 미달이면 ❌
- **기록**: 실패한 실험도 남긴다 — "무엇이 안 되는지"가 자산
